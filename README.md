# 🌳 Генеалогическое Древо (Family Tree)

Добро пожаловать! Это приложение поможет вам создать и визуализировать семейную историю.

---

## 1️⃣ Быстрый старт (для новичков)

Если вы не хотите разбираться в коде, сделайте следующее:

1.  **Скачайте Docker Desktop**: Найдите в Google "Docker Desktop download", установите и запустите его.
2.  **Откройте терминал** (в Windows это Power Shell) в папке с проектом.
3.  **Введите команду**:
    ```bash
    docker-compose up -d
    ```
4.  **Откройте браузер** и перейдите по адресу: [http://localhost:3000](http://localhost:3000)

---

## 2️⃣ Подробный запуск (для разработчиков)

### Требования
*   Node.js 18+
*   PostgreSQL или Docker

### Установка
1.  Установите зависимости:
    ```bash
    npm install
    ```
2.  Настройте файл `.env` (база данных, секретный ключ).
3.  Примените схему базы данных:
    ```bash
    npx prisma db push
    ```
4.  Запустите проект:
    ```bash
    npm run dev
    ```

---

## 3️⃣ Деплой (Как опубликовать в интернет)

Самый простой способ — **Railway** или **Vercel + Supabase**.

### Шаги для Railway (рекомендуется):
1.  Зарегистрируйтесь на [railway.app](https://railway.app).
2.  Нажмите **"New Project"** -> **"Deploy from GitHub repo"**.
3.  Выберите этот репозиторий.
4.  Добавьте базу данных PostgreSQL в проект Railway.
5.  Railway автоматически подставит `DATABASE_URL`.
6.  Добавьте переменную `NEXTAUTH_SECRET` (любая длинная случайная строка).
7.  Готово! Сайт будет доступен по ссылке.

---

## 4️⃣ Если что-то не работает (FAQ)

*   **Порт 3000 занят**: Закройте другие программы (например, другие Node.js процессы) или измените порт в `docker-compose.yml`.
*   **Ошибка базы данных**: Убедитесь, что Docker запущен. Если используете локальную БД, проверьте `DATABASE_URL`.
*   **Белый экран**: Проверьте консоль браузера (F12) на наличие ошибок.

## 5️⃣ «Если вы ничего не понимаете в IT — делайте так»

1.  Установите **Docker Desktop**.
2.  Скачайте этот проект архивом и распакуйте.
3.  Нажмите правой кнопкой на папку проекта -> "Открыть в терминале".
4.  Напишите `docker-compose up` и нажмите Enter.
5.  Подождите 2-3 минуты.
6.  В браузере введите `localhost:3000`.
7.  Для входа используйте: `admin@example.com` / `password123` (после выполнения `npx prisma db seed`).

## Безопасность и Технологии

*   **Next.js**: Современный каркас для быстрых приложений.
*   **Auth**: Защищенный вход через JWT и шифрование паролей (bcrypt).
*   **Prisma**: Надежное управление базой данных.
*   **React Flow**: Красивая и интерактивная визуализация древа.
